# Restart

Restart is a Blender Extension that adds a restart function to Blender's File menu with save/discard prompts and customizable restart behavior.

---
## Features

* File menu restart option and `Ctrl+Alt+Shift+R` keyboard shortcut (customizable in keymaps)
* Confirmation dialog for unsaved changes with *Save*, *Don't Save*, and *Cancel* options
* Automatic "Save As..." dialog handling for new files
* Warning system highlighting unsaved files in red
* Configurable default behavior via addon preferences (*Prompt*, *Save & Restart*, or *Discard & Restart*)
